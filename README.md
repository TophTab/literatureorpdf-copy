# literature-pdf-copy 
## PDF复制（部分实现段间换行符保留）   	
你是否因为复制文献时各种换行而头疼？幸运的是，你找到了这个工具，去除了换行符和各种多出的空格。    
Do you have a headache when copying parts of a document? Fortunately, you've found this tool, removing newlines and all kinds of extra spaces.
